## Usage
Compile with `make`

Add this line to the end of your shell startup file,
such as `.bashrc` or `.zshrc`:<br>
`export PATH="$HOME/path/to/the/repo:$PATH"`

Get the list of available command with `mouli help`

## Update
`git pull && make`
